---

## 📋 페이지 구조 & 스토리라인

> **스토리 흐름**: 공감(Hook) → 문제 공명 → 팀 신뢰 → 검증 근거 → 행동 유도 → 안심

```
1. 헤로 섹션         — Hook: "당신의 이야기가 우리를 움직입니다"
2. 우리가 보는 문제  — 5개 페인포인트 공명 (학부모 "이게 내 문제" 반응 유도)
3. 팀 소개           — "왜 우리가 이 문제에 집착하는가"
4. 우리의 접근 방식  — 어떻게 일하는가 (과정 신뢰)
5. 멘토 네트워크     — 전문가 신뢰 (권위 + 사회적 증명)
6. ASM 소개          — 정부 공식 프로젝트 (공신력)
7. 인터뷰 신청 CTA   — 행동 유도
8. FAQ               — 걱정 해소 (전환 장벽 제거)
9. 풋터              — 연락처 & 안심 정보
```

> ⚠️ **구현 시 주의**: 문서의 섹션 2(팀 소개)와 섹션 3(문제)는 실제 페이지에서 순서를 바꿔야 합니다.  
> 학부모는 "이게 내 문제다" 공명이 먼저 와야 팀 소개를 읽을 동기가 생깁니다.  
> **권장 순서**: 문제 → 팀(왜 우리가 하는가) → 접근방식 → 멘토 → ASM

```
"내 아이의 누적된 기록을 어떻게 읽어야 하고,

```

| 우선 | 랜딩 표현 | 리서치 페인포인트 | 랜딩 역할 |

| 1 | 우리 아이 기록 기준으로 뭘 해야 할지 모름 | ① 생활기록부·성적 기반 개인화 해석 부족 | 핵심 공감 Hook |
| 2 | 고1 첫 90일을 어떻게 준비할지 모름 | ④ 예비고1 준비 방향 공백 | 중3 학부모 전환기 불안 자극 |

#### 1️⃣ 생활기록부·성적 기반 개인화 해석 부족
```
학부모의 말:
"입시 정보는 유튜브, 카페, 책에 넘쳐나요.
 하지만 '우리 아이 생기부를 어떻게 읽어야 하고
 무엇을 해야 할지'는 아무도 설명 안 해줘요"

우리의 관찰:
- 중학교 3년 누적 기록이 쌓여있는데, 그것이 무엇을 의미하는지 모름
- 선택지는 많은데(과목, 활동, 학원), 우선순위가 없음
- 학기마다 고민이 반복됨 (누적되지 않고 초기화됨)

랜딩에서 해야 할 일:
- Hero와 첫 문제 카드에서 가장 먼저 보여주기
- "정보 부족"이 아니라 "내 아이 기록 기준 판단 기준 부족"으로 표현하기
- 인터뷰 신청 이유를 "우리 아이 상황을 설명해보고 싶다"로 만들기
```

#### 2️⃣ 결제 직전 가격·신뢰·개인정보 불안
```
학부모의 말:
"무료 정보는 못 믿겠고, 비싼 상담은 부담돼요.
 게다가 학교 기록을 보내는 것도 불안해요."

우리의 관찰:
- 실제 결제 전환을 막는 것은 필요성 부족이 아니라 신뢰 부족
- 생기부 업로드, 미성년자 정보, 상담자 품질에 대한 불안이 큼
- 무료 정보와 30만 원대 고입 상담 사이에서 비교 행동이 발생함

랜딩에서 해야 할 일:
- 이 페인은 문제 카드보다 FAQ와 신뢰 섹션에서 강하게 처리
- "인터뷰 단계에서는 자료 업로드 필수 아님"을 명시
- 비식별화, 담당자 실명, ASM 공식 프로젝트, 멘토 네트워크를 반복 노출
```

#### 3️⃣ 특목·자사 원서·면접·자소서 우선순위 압박
```
학부모의 말:
"이미 늦은 건 아닌가요?
 원서 전에 뭘 더 할 수 있죠?"

우리의 관찰:

- 고관여 학부모는 30만 원대 상담까지 비교 대상으로 봄

랜딩에서 해야 할 일:
- "특목·자사 준비 중인 분도 환영" 카드로 별도 노출
- 메인 메시지를 특목·자사로 과도하게 좁히지는 않기
- 인터뷰 대상 섹션에서 고관여 사례로 따로 호출하기
```

#### 4️⃣ 예비고1 준비 방향 공백
```
학부모의 말:
"첫 중간고사가 중요하다던데, 이번 방학에 뭘 해야 할까요?
 학원을 더 보내는 게 답인지 모르겠어요."

우리의 관찰:
- 중3 말부터 사교육 지출이 본격화됨
- 학원은 과목 중심으로 답하지만, 학부모의 질문은 학교 선택·생기부 방향·고1 첫 90일 계획에 가까움
- 일반고 선택도 전략이 필요한지 모르는 넓은 불안이 있음

랜딩에서 해야 할 일:
- "예비고1"을 Hero와 신청 대상에 명확히 넣기
- "첫 90일", "학교 선택", "생기부 방향"처럼 시간 압박이 있는 문구 사용
- 학원 대체가 아니라 학원 선택 전 의사결정 보조로 포지셔닝
```

#### 5️⃣ 담임·진로 상담 질문 준비 부족
```
학부모의 말:
"상담 시간은 짧은데, 뭘 물어봐야 할지 모르겠어요.
 그냥 듣고만 오면 아까울 것 같아요."

우리의 관찰:
- 학교 상담은 신뢰도가 높고 무료라 강력한 대체재
- 문제는 상담 자체가 아니라 상담 전에 기록과 질문을 정리하지 못하는 것
- 무료 체크리스트 → 개인화 리포트 → 전문가 검수로 이어지는 진입 경로가 자연스러움

랜딩에서 해야 할 일:
- 학교 상담을 대체한다고 말하지 않기
- "상담 전 질문 정리"를 낮은 부담의 첫 가치로 제시
- 인터뷰 CTA를 "내 고민을 정리해보기"처럼 가볍게 만들기
```

```

  섹션 제목: "많은 학부모님이 이렇게 말씀하셨습니다"
  ↓
  Pain Point 카드 (2~3열 그리드)
  각 카드: 아이콘 + 제목 + 실제 발화문 인용
  ↓
  클로징 메시지: "이것이 우리가 해결하려는 문제입니다"
  ↓
  2차 CTA (작게)
```

#### 팀 소개 섹션
```
[배경: #F8F9FA 라이트 그레이]

  팀원 카드 3개 (수평 or 2+1)
  각 카드:
  - 실제 사진 필수 (원형, 120x120px)
  - 이름 + 역할
  - 핵심 배경 2줄
  - "이 팀에 있는 이유" 인용 (1줄)
```

#### 멘토 섹션
```
[배경: 따뜻한 화이트]

  공식 담당 멘토 2명 카드 (나란히)
  ↓
  비기술 멘토 카드 (구분된 스타일)
  ↓
  로고 스트립: 삼성전자, 모비딕벤처스, KIST 로고
```

#### ASM 신뢰 섹션
```
[배경: 딥 네이비, 흰 텍스트]

  숫자 4개로 신뢰 표현 (큰 폰트)
  "16년 역사 | 17기 450명 선발 | 2,000명+ 수료생 | 팀당 1,200만원"
  ↓
  과학기술정보통신부 로고
```

#### CTA 섹션
```
[배경: 오렌지 그라디언트]
[중앙 정렬]

  "50분으로 시작할 수 있습니다"
  ↓
  버튼 (흰 텍스트, 딥 네이비 배경)
  ↓
  인센티브 + "개인정보 보호" 한 줄 메모
```

### 📱 모바일 최적화 (최우선)

```
Sticky 하단 CTA (스크롤 200px 이후 노출):
  "인터뷰 신청하기 → " 고정 버튼
  X 버튼으로 닫기 가능

카카오 공유 버튼:
  Hero 섹션 포함 (카카오 노란 버튼)
  "지인에게 공유하기" 유도

이미지 최적화:
  WebP 포맷, lazy loading
  팀 사진·멘토 사진 필수 (텍스트만 있으면 신뢰도 낮음)
  사진 없을 시 일러스트 아바타로 대체
```

### 🛡️ 신뢰 신호 (Trust Signals) 배치

```
위치             신뢰 요소
─────────────────────────────────────────────────────
Hero 상단      "AI·SW마에스트로 17기 선발 과정 통과" 뱃지
팀 섹션        실명 + 실제 경력 (교육 서비스, KOPLE)
멘토 섹션      소속 기관 로고 (삼성, 모비딕벤처스, KIST)
ASM 섹션       과학기술정보통신부 로고 + 16년 역사 + 17기 450명 선발
FAQ 섹션       "개인정보는 어떻게 처리되나요?" 명시
Footer         담당자 실명 + 이메일 (박세종)
```

### ⚠️ 학부모 타겟 UX 주의사항

```
❌ 피해야 할 것:
  - 너무 트렌디한 디자인 (부모 세대 친화 아님)
  - 영문 용어 과다 (MVP, CTA, LTV 등 — 한국어로)
  - 긴 텍스트 블록 (학부모는 스캔하며 읽음)
  - 팝업/모달 (신뢰감 저하)
  - 자동 실행 영상·사운드

✅ 해야 할 것:
  - 실제 학부모 발화문 인용 (따옴표 형식)
  - 숫자로 말하기: 100+명, 3만 건, 20년, 2,700명
  - "50분, 무료, 비대면 가능" 반복 노출
  - 담당자 실명 + 연락처 명확히
  - CTA 버튼 3회 이상 배치 (Hero, 문제 섹션 후, 팀 섹션 후)
```

### 🧪 A/B 테스트 우선순위

```
1순위: Hero 헤드라인
   A: "당신의 이야기가 우리를 움직입니다"
   B: "학부모님의 고민, 50분만 들려주세요"

2순위: CTA 버튼 문구
   A: "인터뷰 신청하기 (50분)"
   B: "내 고민 들려주기 (문화상품권 제공)"

3순위: Hero 배경
   A: 딥 네이비
   B: 따뜻한 화이트 + 딥 네이비 텍스트
```
