# 기존 코드 재사용 결정 사항

## 1. 팀 프로필 사진
- **결정**: 정장 사진으로 교체 (신뢰성 강화)
- 원본 저장 위치: `src/assets/images/team/` (JPG 또는 PNG)
- WebP 변환 후 위치: `src/assets/images/team/optimized/` (기존 파일명 덮어쓰기)
  - `sejong-profile.webp`
  - `junhyeok-profile.webp`
  - `semin-profile.webp`

## 2. Reveal 컴포넌트 (스크롤 애니메이션)
- **결정**: 그대로 재사용
- 이유: `useReducedMotion` 지원, 성능 문제 없음, 새로 구현할 이유 없음

## 3. 카카오 지도
- **결정**: 도입하되, "연수센터" 명칭 노출 제거
- 표시 방식: 건물명은 "포스트타워 7층"으로만 표기. "AI·SW마에스트로 연수센터"라는 공식 명칭은 UI에 노출하지 않음
- 이유: 학부모에게 "연수생 팀" 인상을 줄 수 있음

## 4. 멘토 프로필
- **결정**: 도입하되, 팀원보다 작은 비중으로 표시
- 팀원 카드보다 축소된 레이아웃 (사진 없음, 이름+역할+한 줄 요약 수준)
- 회사 페이지(team310.kr)에만 포함, 리서치 랜딩에는 미포함

## 5. 유튜브 영상
- **결정**: 도입
- 회사 페이지에 포함. 기존 `youtube-nocookie.com` 프라이버시 처리 유지

## 6. 일하는 원칙 (Work Principles)
- **결정**: 보류
- 학부모 입장에서 필요성 불명확, 추후 검토
