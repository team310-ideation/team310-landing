1. 기관·학교 협력 컨택 시 보여줄 신뢰의 매개체
2. 잠재 협력자·미래 채용자에게 팀 정체성 전달
판매·전환이 아닌 "이 팀이 진짜인지"를 짧은 시간에 증명하는 페이지.

- 2차: 잠재 채용자, 협력자, 투자자
- 공통: "스타트업 허세"에 대한 경계심 있음 — 과한 비전 선언이나 미래형 카피는 역효과

1. Hero
   - 팀명 "팀 310" 크게
   - 한 줄 미션: "중3 가정의 진로·입시 페인포인트를 연구합니다"
   - CTA: "현재 진행 중인 리서치 보기 →" (랜딩페이지 /research 링크)
   - 과한 일러스트나 추상 그래픽 X. 여백 중심.

2. Origin Story
   - 학부모의 실제 질문 인용으로 시작
   - 4~5단락 텍스트, 좌측 정렬, 가독성 우선
   - 인용구는 시각적으로 강조 (큰 따옴표 + 컬러 액센트)

3. 팀명의 의미 ("왜 310인가요?")
   - 별도 섹션, 어두운 배경 또는 카드형 컨테이너로 구분
   - 큰 숫자 시각화: "10년 안에 300명의 일자리"
   - 짧은 부연 단락 2~3문장

4. 팀 멤버
   - 가로 3개 카드 (데스크톱), 세로 스택 (모바일)
   - 각 카드: 사진 자리(임시 이니셜 원형) + 이름 + 역할 한 줄 + 핵심 경력 2~3줄
   - 카드 간 동일 높이, 사진 자리는 정사각형 또는 원형 통일

5. 현재 진행 중인 리서치
   - 랜딩페이지로 자연스러운 게이트웨이
   - "지금 우리는 중3 학생 10명, 학부모 30명을 만나고 있습니다" 류
   - CTA 박스: 리서치 페이지 링크

6. 함께하기 (연락처)
   - 3개 분리된 이메일 카드: 기관·학교 협력 / 팀 합류 / 일반 문의
   - 클릭하면 mailto: 링크

7. 푸터
   - 팀명, 대표 이메일, 개인정보 처리방침 링크
   - 사업자 정보는 빈 칸으로 두기 (등록 후 추가 예정)
